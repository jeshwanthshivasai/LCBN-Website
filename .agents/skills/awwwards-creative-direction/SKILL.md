---
name: awwwards-creative-direction
description: High-level creative direction rules for an Awwwards-tier site
---

1. **The Hero is Everything**: The first 3 seconds determine if a user stays. The hero section needs to immediately establish the aesthetic (dark mode, sleek gradients/3D, perfect typography, brand color `#96CC39`).
2. **Consistency**: The same custom ease, the same hover interactions on EVERY. SINGLE. BUTTON.
3. **Pacing**: Don't throw all animations at the user at once. Sequence text parsing, then image reveals, then layout shifts.
4. **Cinematography**: Apply film rules (depth of field, bokeh, lighting, post-processing grain) if using any 3D or video components on the site.
